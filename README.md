# Dotfiles Setup 🚀

## Steps

1. Clone this repo.

```zsh
# SSH
git clone git@github.com:gordos-aron/dotfiles.git ~/dotfiles

# HTTPS
git clone https://github.com/gordos-aron/dotfiles.git ~/dotfiles
```

2. Create a symlink in the Home directory.

```zsh
ln -s ~/dotfiles/.gitconfig ~/.gitconfig
```
